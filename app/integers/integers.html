Largest Integer

  1. Implement a Javascript function that takes any comma delimited string as a parameter and returns the LARGEST integer value. The user will enter the string in the text box below and the result will display in the panel. If the string does not return any integers, "No Integers Found" should be displayed. Sample I/O:
    • input: "10, 3, 232, ,3000, 120, B, 51, 6, 7, 120, 10, 10" - output: "3000"
    • input: "A, B, C" - output: "no integers found"
    • input: "500, 11, 222, 44 , 6 , 7, 22,Z," - output: "500"
    • input: "" - output: "no integers found"
    • input: "-3, 0, -1" - output: "0"
  2. Make the text box required so that the user cannot click on the Submit button if there is no value.
  3. Use the class panel-danger to make the results panel red if the result is "No Integers Found".
  4. Write unit tests to validate your controller. Execute npm test to run the unit tests.
